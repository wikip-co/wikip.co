## Description

Amazon Virtual Private Cloud (Amazon VPC) enables you to launch Amazon Web Services resources into a virtual network you've defined. This virtual network resembles a traditional network that you'd operate in your own data center, with the benefits of using the scalable infrastructure of AWS.[^1]

[^1]: https://docs.aws.amazon.com/toolkit-for-visual-studio/latest/user-guide/vpc-tkv.html
